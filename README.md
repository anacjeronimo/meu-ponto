# 📱 Meu Ponto

Meu Ponto é um aplicativo desenvolvido em Flutter para facilitar o armazenamento e a consulta de comprovantes de ponto impressos.

## 🎯 Objetivo

O projeto surgiu da necessidade de guardar comprovantes emitidos por relógios de ponto, evitando a perda desses registros e permitindo uma consulta rápida diretamente pelo celular.

## 🚀 Tecnologias

- Flutter
- Dart
- SQLite
- Google ML Kit (OCR)
- Figma
